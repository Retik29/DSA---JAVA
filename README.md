# 🚀 DSA with Java — Data Structures & Algorithms Implementation

Welcome to the **DSA-JAVA** repository! This project is a comprehensive collection of Data Structures and Algorithms implemented in Java, ranging from fundamental concepts to advanced problem-solving techniques.

## 📂 Project Structure

The codebase is organized into modular directories based on topics:

| Module | Description |
| :--- | :--- |
| 📊 **Arrays** | Basic operations, sliding window, prefix sums, and two-pointer techniques. |
| 🔍 **Searching** | Binary search, finding occurrences, and optimization problems. |
| ⚡ **Sorting** | Bubble, Selection, Insertion, Merge, Quick, and specialized sorts. |
| 🔗 **LinkedList** | Singly, Doubly, and Circular Linked Lists implementations & problems. |
| 🥞 **Stack & Queue** | Balanced parentheses, next greater element, and queue implementations. |
| 🌳 **Trees & BST** | Traversals, height/diameter, LCA, and Binary Search Tree operations. |
| 🕸️ **Graph** | BFS, DFS, Cycle Detection, and shortest path algorithms. |
| 🧠 **Dynamic Programming** | Optimal substructure and overlapping subproblems solutions. |
| 🎯 **Greedy** | Huffman coding, Fractional Knapsack, and Activity Selection. |
| 🔄 **Recursion** | Core recursive patterns and classic puzzles. |
| 🛠️ **Bits & Math** | Bitwise manipulation and essential mathematical algorithms. |
| 📝 **Strings** | Pattern matching, anagrams, and string transformations. |

## ✨ Key Features

- **Concise & Readable**: Clean implementations with helpful comments.

- **Topic-Wise Categorization**: Easy to navigate and learn specific modules.

## 🚀 How to Run

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Retik29/DSA---JAVA.git
   ```

2. **Navigate to a Directory**:
   ```bash
   cd DSA---JAVA/Arrays
   ```

3. **Compile and Run**:
   ```bash
   javac Operations.java
   java Operations
   ```
